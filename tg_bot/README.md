# 🚗 Auction Car Bot — Copart & IAAI

Telegram-бот, який парсить лоти з аукціонів Copart та IAAI і повертає красивий колаж з фото + повну інформацію про авто.

---

## 📦 Встановлення

### 1. Клонуй/розпакуй проект

```bash
cd tg_bot
```

### 2. Встанови залежності

```bash
pip install -r requirements.txt
```

### 3. Отримай токен бота

1. Відкрий Telegram → [@BotFather](https://t.me/BotFather)
2. Надішли `/newbot` і слідуй інструкціям
3. Скопіюй токен

### 4. Встав токен

**Варіант A — через змінну середовища (рекомендовано):**
```bash
export BOT_TOKEN="1234567890:ABCdef..."
python bot.py
```

**Варіант B — прямо в коді:**
У файлі `bot.py` знайди рядок:
```python
BOT_TOKEN = os.getenv("BOT_TOKEN", "YOUR_BOT_TOKEN_HERE")
```
і заміни `YOUR_BOT_TOKEN_HERE` на свій токен.

---

## 🚀 Запуск

```bash
python bot.py
```

---

## 🤖 Як використовувати

1. Знайди лот на [copart.com](https://copart.com) або [iaai.com](https://www.iaai.com)
2. Скопіюй посилання на сторінку лоту
3. Надішли посилання боту
4. Отримай колаж з фото + всю інформацію

**Приклади посилань:**
- `https://www.copart.com/lot/12345678`
- `https://www.iaai.com/vehicle/12345678`

---

## 📸 Що повертає бот

- Колаж до 9 фото у сітці 3×3
- Марка, модель, рік
- VIN-код
- Поточна ставка
- Пошкодження (primary/secondary)
- Двигун, трансмісія, паливо
- Колір, тип кузова
- Пробіг
- Локація аукціону
- Номер лоту та дата продажу
- Тип документів

## 🛠 Технічний стек

| Компонент | Бібліотека |
|-----------|------------|
| Telegram Bot | `python-telegram-bot` |
| Парсинг сторінок | `requests` + `beautifulsoup4` |
| Генерація колажу | `Pillow` |

---

## ⚠️ Примітки

- Copart використовує публічне API (`/public/data/lotdetails/`) — це легально
- IAAI парситься через HTML
- Бот не зберігає жодних даних
- Фото беруться з CDN аукціонів у найвищій якості
